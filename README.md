## 사용처 🧩
초・중학생들이 쉽게 안드로이드 앱 개발을 배울 수 있도록 블록코딩 기반으로 교육하였다.

<br>

## 캠프 요약 📃
#### [1일차 (1-2차시)](./lectures/day1/day1.md)
- 변수 이해하기
- 앱인벤터 사용법 익히기 
#### 2일차 (3-4차시)
- 반복문과 조건문 이해하기
- 앱 따라하기 기초 
#### 3일차 (5-6차시)
- TTS 이해하기 
- 앱 따라하기 심화
#### 4일차
- 미니 앱인벤터 해커톤

<br>

## 캠프 목표 ✨

1. 앱인벤터를 사용하여 스스로 설계한 앱을 만들 수 있다.
2. 프로그래밍에 필요한 기초 개념을 이해할 수 있다.
