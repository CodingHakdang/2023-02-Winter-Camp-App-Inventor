## 캠프 정보 🔖
**교구** : MIT App Inventor 2

**기간** : 2월6일~2월9일 (일 차당 4h, 4일 차 4h30m)

**강사 수** : 메인강사 2명 보조강사 13명\
**수강생 수** : 총 15명 (9-15세)

**주최** : ifez 인천경제자유구역\
**주관** : 글로벌문화예술교육단체, 코딩학당, KRIS국제학교\
**후원기업** : GFX, GENBURGER

**캠프 장소** : 인천 송도 IFEZ G-Tower

## 사용목적 🧩
초・중학생을 대상으로 블록 코딩을 이용해 안드로이드 앱을 개발하는 데 사용하였습니다.
