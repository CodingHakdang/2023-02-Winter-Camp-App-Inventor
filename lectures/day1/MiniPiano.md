<br>

## Event

```
🖼 ImageSprite Event
```

> **흰 건반을 터치 다운 했을 때** \
> 흰 건반 이미지를 눌린 건반 이미지로 바꾼다. \
> 해당 음계의 소리를 낸다.

> **흰 건반을 터치 업 했을 때** \
> 눌린 흰 건반 이미지를 흰 건반 이미지로 바꾼다.
