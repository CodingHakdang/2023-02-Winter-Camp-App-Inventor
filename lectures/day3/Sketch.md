## 설계 목표

캔버스 컴포넌트 사용법을 학습한다. \
슬라이더 센서 사용법을 학습한다.

<br>

## 설명 

터치를 통해 화면에 그림을 그릴 수 있는 앱 \
선의 색상과 굵기를 조절할 수 있다.

<br>

## 이벤트 

```
⏹ Button Event
```

> **지우개 버튼을 클릭했을 때**\
> 화면을 초기화한다.

> **색상 버튼을 클릭했을 때**\
> 선의 색상이 바뀐다.

```
📡 Sensor Event
```

> **슬라이더를 드래그했을 때**\
> 선의 굵기가 바뀐다.
