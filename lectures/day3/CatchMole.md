## Description

화면을 터치하여 두더지를 잡는 게임

<br>

## Event

*Screen 1*

```
⏹ Button Event(1)
```

> **다시하기 버튼을 클릭했을 때** \
> 점수 및 타이머를 초기화한다.

> **두더지를 터치했을 때** \
> 잡기성공 점수가 1 증가한다.

> **배경을 터치했을 때** \
> 잡기실패 점수가 1 증가한다.

```
📡 Sensor Event(1)
```
> **타이머가 작동할 때** \
> 두더지가 랜덤 위치로 이동한다.

> **타이머가 0일 때** \
> Screen 2 로 이동한다. \
> 점수를 Screen 2 로 전달한다.

*Screen 2*

```
📡 Sensor Event(2)
```

> **Screen 2 가 초기화되었을 때** \
> 점수를 화면에 표시한다.

```
⏹ Button Event(2)
```

> **다시하기 버튼을 클릭했을 때**\
> Screen 1 로 이동한다.
