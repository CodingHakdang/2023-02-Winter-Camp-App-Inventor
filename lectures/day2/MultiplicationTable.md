## 설계 목표

반복문의 개념을 이해한다. \
스크린이 이동하는 방법을 이해한다.

<br>

## Description

1-9 중 원하는 단수를 입력하면, 다른 스크린에서 해당 단수를 출력한다.

<br>

## Event

> **확인 버튼을 클릭했을 때** \
> 입력받은 텍스트값을 매개변수로 전달하고 Screen2를 연다. 

> **Screen2가 초기화 되었을 때** \
> Screen1에서 받은 매개변수 숫자를 출력한다. 
> number 변수는 1-9까지 1씩 증가하며 반복문을 돌게 만든다. 
> Screen1에서 받은 매개변수와 반복문에서 초기화한 number를 곱하여 결과값을 출력한다. 

> **다시하기 버튼을 클릭했을 때** \
> Screen1으로 스크린을 이동한다.
